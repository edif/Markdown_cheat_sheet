## Text Formatting

```markdown
*italic*
**bold**
***bold-italic***
++Underline++
~~Strikethrough~~
==Highlight==
^Superscript^
~Subscript~
```

## Lists

```markdown

1. First ordered list item
2. Another item

   - Unordered sub-list.

3. Actual numbers don't matter, just that it's a number

   3.1 Ordered sub-list

4. And another item.

* Unordered list can use asterisks
- Or minuses
+ Or pluses

```
